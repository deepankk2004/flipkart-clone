# Flipkart Clone – Full Stack E-Commerce Web Application

## Project Description
This project is a full-stack e-commerce web application inspired by Flipkart. The application allows users to browse products, view product details, add items to the cart, and place orders. The project replicates Flipkart's UI design and basic e-commerce functionality.

---

## Features
- Product listing page (Home page)
- Product detail page
- Add to Cart functionality
- Cart page with quantity management
- Checkout page
- Order success page
- Flipkart-style responsive UI
- Backend API for products and cart
- Product image support
